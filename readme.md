-- Tecnologias --
Este projeto web PHP é um protótipo do sistema.
O método de cálculo e a disposição das informações não se dão por forma dinâmica por se tratar de um protótipo.
Realiza conexão com banco de dados MySQL.
Foi utilizado:
- PHP 7
- HTML 5
- CSS (Bootstrap)
- JavaScript (JQuery)
- MySQL
- Apache
- PHPMyAdmin

-- Informações úteis --

Necessário um servidor/container para execução da aplicação. Para desenvolvimento, foi utilizado o XAMPP (Apache) (https://www.apachefriends.org/pt_br/index.html).

O arquivo SQL para importar o banco e as tabelas se encontra no caminho resources/sart_script_criacao_banco.sql

Dados de conexão padrões do MySQL do XAMPP:
- Usuário: root
- Senha: <vazia>
- Base de Dados: sart
Estas configuracoes podem ser alteradas no caminho content/configuracao/conectabd.inc.php
